PIMP MY CAUSE
Women Hack for Non Profits is a program for women in tech to write open source software for non profit organisations. One such organisation approached WHFNP to rebuild their website a company called 'Pimp My Cause' who provide volunteers the opportunity to support good causes while developing their marketing skills

In my capacity as front end developer on the Pimp My Cause project I built the front page according to specification while also adapting the design for responsive behaviour. The technologies I used were Zurb Foundation framework, HTML5, CSS3, Sass and Grunt to run tasks.

I worked within a team of women to produce a total solution for the client with regular client meetings. The project was managed using JIRA and a SCRUM approach with weekly stand ups. The demands and work practices comparable to any such industry based project provided me with real world experience as a Front End Developer.

FOOF RECORDS
Foof Records is a London based record label providing a place to discover new independent artists and exclusive pay-what-you-want downloads. Foof support their artists above all else enabling creative freedom without contract restraints while promoting music for the enjoyment of their audience.

My involvement in the Foof Records website was the design and layout of the site along with development of the static site. I am currently building a WordPress site to support this custom theme. The technologies I used for design were Illustrator/Photoshop and for frontend build a proprietary framework called Matter along with HTML5, CSS3, LESS and Grunt.

During this project I worked with the client to scope out their requirements and preferred style of website. Meeting and liaising throughout the process and then providing post care support.

CROOKED
Crooked is a ficticious organisation selling skateboards and clothing. A site currently under construction to demonstrate ecommerce with product specifications, add to basket functionality and customer reviews. This site also provides a blogging platform along with social media sharing.

The Crooked website will require much the same technologies and approach as this personal website has taken to build. The project scaffolding will be the same using Bower to install frontend tools like Bootstrap and Font Awesome. The Sass preprocessor will be used and Gulp for task automation. The static site will be built according to the PSD design and will then be used to built a WordPress site.

This site will not be built for a customer though is expected to have functionality that will create transferrable skills for other such sites.
